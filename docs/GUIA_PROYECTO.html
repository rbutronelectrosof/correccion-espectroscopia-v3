GUIA DEL PROYECTO
Sistema de Clasificacion Espectral Estelar

1. DESCRIPCION GENERAL

Este sistema clasifica automaticamente estrellas segun su tipo espectral (O, B, A, F, G, K, M) analizando sus espectros de luz. Utiliza multiples metodos de clasificacion combinados para lograr una precision del 86.5%.

3. COMO FUNCIONA EL SISTEMA

Pipeline de Procesamiento

ENTRADA: Archivo de espectro (.txt o .fits)

↓

1. NORMALIZACION
Ajuste del continuo + Sigma-clipping

↓

2. MEDICION DE LINEAS
Detecta 50+ lineas espectrales
Calcula anchos equivalentes

↓

3. CLASIFICACION MULTI-METODO

Clasificador Fisico (reglas astronomicas)10%
Arbol de Decision (scikit-learn)40%
Template Matching (comparacion)10%
KNN / CNN (redes neuronales)40%

↓

4. VOTACION PONDERADA
Combina resultados + Calcula confianza

↓

SALIDA: Tipo espectral + Confianza (%)
Ejemplo: "G2V" con 87% confianza

5. INSTRUCCIONES DE USO

Paso 1: Instalacion
Ejecutar: INSTALAR.bat
Instala: numpy, scipy, matplotlib, pandas, scikit-learn, flask, astropy
Paso 2: Entrenamiento (opcional, ya hay modelos entrenados)
Ejecutar: ENTRENAR.bat
Entrena los modelos con el catalogo ELODIE (860 espectros)
Paso 3: Usar la aplicacion
Ejecutar: INICIAR.bat
Abre el navegador en http://localhost:5000

Uso de la interfaz web:

  1. Subir archivo de espectro (.txt o .fits)
  2. Ver resultado de clasificacion
  3. Examinar grafico con lineas detectadas
  4. Exportar resultados a CSV
